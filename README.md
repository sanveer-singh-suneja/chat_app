# Chatify 💬

A real-time one-to-one chat application built with Flutter and Firebase.

![Flutter](https://img.shields.io/badge/Flutter-3.x-blue)
![Firebase](https://img.shields.io/badge/Firebase-Latest-orange)
![Material 3](https://img.shields.io/badge/Material-3-green)

---

## Features

- **Authentication** — Register, login, logout with Firebase Auth
- **Real-time messaging** — Instant text and image messages via Firestore
- **User discovery** — Browse and search all registered users
- **Online status** — See who's online and when others were last seen
- **Dark mode** — Toggle between light and dark theme, persisted locally
- **Profile screen** — Update profile picture stored in Firebase Storage
- **Delete messages** — Long press your own messages to delete them
- **Image messages** — Send images from your gallery

## Firebase Setup (Required)

### Step 1 — Create a Firebase Project

1. Go to [Firebase Console](https://console.firebase.google.com/)
2. Click **Add Project**
3. Name it `chatify` (or anything you like)
4. Disable Google Analytics if you don't need it
5. Click **Create Project**

---

### Step 2 — Enable Authentication

1. In Firebase Console → **Authentication** → **Get started**
2. Go to the **Sign-in method** tab
3. Enable **Email/Password**
4. Click **Save**

---

### Step 3 — Create Firestore Database

1. In Firebase Console → **Firestore Database** → **Create database**
2. Choose **Start in test mode** (for development)
3. Select a Cloud Firestore location
4. Click **Enable**

After setup, copy the Firestore rules from `firestore.rules` and paste them into the Firestore Rules tab.

---

### Step 4 — Enable Firebase Storage

1. In Firebase Console → **Storage** → **Get started**
2. Accept the default rules for now and click **Done**

After setup, copy the rules from `storage.rules` and paste them into the Storage Rules tab.

---

### Step 5 — Add Your App to Firebase

#### Android

1. In Firebase Console → **Project Overview** → **Add app** → Android icon
2. Package name: `com.example.chatify`
3. Click **Register app**
4. Download `google-services.json`
5. Replace `android/app/google-services.json` with the downloaded file

#### iOS (optional)

1. Click **Add app** → iOS icon
2. Bundle ID: `com.example.chatify`
3. Download `GoogleService-Info.plist`
4. Place it in `ios/Runner/` using Xcode (drag and drop)

---

### Step 6 — Configure FlutterFire

Install the FlutterFire CLI and run:

```bash
dart pub global activate flutterfire_cli
flutterfire configure
```

This auto-generates `lib/firebase_options.dart` with the correct values for your project.

> **Note:** The existing `firebase_options.dart` has placeholder values — replace them using the CLI command above.

---

### Step 7 — Run the App

```bash
flutter pub get
flutter run
```

## Firestore Data Structure

```
users/
  {uid}/
    uid: string
    name: string
    email: string
    profileImage: string (URL)
    isOnline: boolean
    lastSeen: timestamp
    createdAt: timestamp

chats/
  {chatId}/             ← chatId = sorted(uid1, uid2) joined by "_"
    participants: [uid1, uid2]
    messages/
      {messageId}/
        messageId: string
        senderId: string
        receiverId: string
        message: string
        imageUrl: string
        messageType: "text" | "image"
        timestamp: timestamp
```

## Known Limitations

- No push notifications
- No read receipts
- No message editing
- No group chats
- No typing indicators
- Images are not compressed before upload on iOS (Android uses imageQuality: 60)
